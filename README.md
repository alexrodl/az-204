# AZ-204 - Developing Solutions for Microsoft Azure

Bienvenidos al curso AZ-204 🚀

## 📌 Objetivo
Este curso está orientado a desarrolladores que quieren aprender a construir soluciones en Azure y prepararse para el examen de certificación AZ-204.

## 📚 Topic Areas

### 🧩 Compute
- Azure App Service
- Azure Functions
- Azure Container Services

### 💾 Data
- Azure Storage
- Azure Cosmos DB

### 🔌 Integration
- Azure API Management
- Events and Messaging

### 🔐 Security
- Authentication & Authorization
- Secure Solutions

### 📊 Monitoring
- Application Insights

## 🗂 Estructura del repositorio

- `/presentations` → Material de clase
- `/labs` → Laboratorios prácticos
- `/demos` → Ejemplos en vivo
- `/resources` → Links útiles
- `/setup` → Configuración inicial

## ⚙️ Requisitos
- Cuenta de Azure
- Visual Studio / VS Code
- .NET 8 SDK

## 🚀 Recomendación
Seguir cada clase en orden y realizar los laboratorios.
