# AZ-204 - Developing Solutions for Microsoft Azure

Bienvenidos al curso AZ-204 🚀

## 📌 Objetivo
Este curso está orientado a desarrolladores que quieren aprender a construir soluciones en Azure y prepararse para el examen de certificación AZ-204.

## 📚 Topic Areas

### 🧩 Compute
- Azure App Service
- Azure Functions
- Azure Container Services

### 💾 Data
- Azure Storage
- Azure Cosmos DB

### 🔌 Integration
- Azure API Management
- Events and Messaging

### 🔐 Security
- Authentication & Authorization
- Secure Solutions

### 📊 Monitoring
- Application Insights

---

## 💻 Herramientas de desarrollo

- Visual Studio Code --> https://code.visualstudio.com/download 
- .NET 8 SDK 
    
    Instalar
    ```powershell
    winget install Microsoft.DotNet.SDK.8
    ```

    Checkear versiones instaladas
    ```powershell
    dotnet --list-sdks
    ```
- ⚡ Azure Functions Core Tools
    ```powershell
    winget install Microsoft.Azure.FunctionsCoreTools
    ```
---

## 🔌 Extensiones de VS Code

Instalar las siguientes extensiones:

- Azure Functions
- C# Dev Kit
